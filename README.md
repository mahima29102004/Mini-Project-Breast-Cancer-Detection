# Breast Cancer Detection Using CNN and MLP

## Description
This project detects breast cancer using two models: CNN and MLP.

## Methodology 1
MobileNetV2 CNN model is used for histopathology image classification.

## Methodology 2
MLP (Multi-Layer Perceptron) model is used for clinical dataset classification.

## Evaluation Metrics
Accuracy, AUC Score, Confusion Matrix, ROC Curve, Precision-Recall Curve.

## Technologies Used
Python, TensorFlow, Keras, Scikit-learn, NumPy, Pandas, Matplotlib, Seaborn.
